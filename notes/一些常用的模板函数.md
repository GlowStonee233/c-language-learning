# 1.计算一个int a的int N次幂
```c
int i_pow(int a,int N)
{
int p=1，i;
for(i=0;i<N;i++)
{
p=p*a;//易错点
}
return p;
}
```

以下程序以二进制为例
```c
void to_binary(unsigned long n)
{
	int r = n % 2;
	if (n >= 2) 
	{
		to_binary(n / 2);
	}
	putchar(r == 0 ? '0' : '1');
}
```

# 5.交换两个int整数
```c
void interchange(int *a,int *b)
{
int temp;
temp=*a;
*a=*b;
*b=temp;
}
```
调用时输入
```c
interchange(&x, &y);
```
